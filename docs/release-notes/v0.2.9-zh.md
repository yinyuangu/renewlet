# Renewlet v0.2.9

Renewlet v0.2.9 让 Docker 和 Cloudflare 下的主页面切换更稳，修复切页时偶发的空白和闪黑。

[English ->](./v0.2.9-en.md)

---

## 升级前请先读

Cloudflare 一键部署用户仍然从生成仓库运行 `Sync Renewlet Upstream` 升级，不要重新点一键部署按钮。旧生成仓库如果还没有这个入口，按 [Cloudflare 部署文档](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.zh-CN.md#旧一键部署用户) 一次性补上。

---

## 改进

- 仪表盘、订阅、日历、统计和设置之间切换更顺，慢网络下也不再突然空白。
- 统计页刷新汇率时会保留当前内容，不再因为请求慢而整页退回加载状态。

## 修复

- AI 识别草稿自动补续费日期时，不再把仍需要用户确认的其他字段一并确认。
