# Renewlet v0.2.0

Renewlet v0.2.0 把语言选择和 AI 识别输出收拢到同一套用户语言规则。首次进入时会按浏览器或请求语言显示界面，保存后的语言偏好跟随账号；AI 识别只让新生成的备注、分类、标签和支付方式跟随当前语言，不改用户输入或已有配置原文。设置页和统计页也修掉了几个容易打断使用的界面细节。

[English ->](./v0.2.0-en.md)

---

## 优化改进

- 首次进入会按浏览器或请求语言选择界面；没有匹配语言时默认英文。
- 设置页里的语言切换只做即时预览，点击保存后才写入账号并跨设备生效。
- AI 识别会按当前语言生成新的备注、分类、标签和支付方式，同时保留用户输入和已有配置原文。
- 页头、设置页分区、登录、初始化和重置密码页面在窄屏下更少出现遮挡或溢出。
- 统计页“按个人成本统计”入口移到总览标题旁，加载占位也同步到相同位置。

---

## Bug 修复

- 修复 AI 识别可能被示例语言带偏，导致生成内容不跟随当前界面语言的问题。
- 修复设置页目录在桌面端跳转后，目标分区贴得离顶部页头过近的问题。
- 修复统计页加载完成后个人成本开关换位置造成的视觉跳动。
