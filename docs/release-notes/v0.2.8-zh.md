# Renewlet v0.2.8

Renewlet v0.2.8 重点优化 Cloudflare 部署在订阅较多时的速度和可靠性。订阅列表、公开 API、提醒、自动续订和云备份排程更稳，私有 Logo 加载也更顺滑。

[English ->](./v0.2.8-en.md)

---

## 升级前请先读

Cloudflare 一键部署用户仍然从生成仓库运行 `Sync Renewlet Upstream` 升级，不要重新点一键部署按钮。旧生成仓库如果还没有这个入口，按 [Cloudflare 部署文档](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.zh-CN.md#旧一键部署用户) 一次性补上。

---

## 改进

- Cloudflare 版订阅列表在筛选、搜索、标签和总数统计时响应更快。订阅较多时，翻页和公开 API 总数更稳定。
- Cloudflare 版通知、自动续订和云备份排程更节省资源，只处理当前需要处理的账号；停机或失败后仍会补上错过的任务。
- 私有 Logo 和图标会在同一登录会话中复用已加载图片，列表重排、分页或打开弹窗时减少重复请求和闪烁。
- 订阅列表请求会短时间复用缓存，返回页面或切换视图时不再频繁重新拉取相同数据。
- 统计页费用走势 tooltip 去掉重复说明行，订阅明细更紧凑。
