# Renewlet v0.2.7

Renewlet v0.2.7 makes the subscription list easier to narrow down. You can combine billing cycle, payment method, currency, renewal date, pinned state, public page visibility, and reminder filters instead of scrolling through a long ledger.

[中文 ->](./v0.2.7-zh.md)

---

## Read Before Upgrading

Cloudflare one-click deploy users should still upgrade from the generated repository by running `Sync Renewlet Upstream`. Do not click the one-click deploy button again. If an older generated repository does not have this entry yet, add it once from the [Cloudflare deploy documentation](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.md#existing-one-click-deploy-users).

---

## New Features

- The subscription list now has More filters for combining billing cycle, payment method, currency, renewal date range, pinned state, public page visibility, reminder state, and repeat reminders.
- Payment method and currency filters support search, multi-select, and selection previews, so long option lists are easier to use.

---

## Improvements

- Active filters now appear in one place, where each one can be removed on its own or cleared all at once.
- Filtered totals, pagination, and results stay consistent across Docker and Cloudflare deployments, making long subscription lists more reliable to browse.
