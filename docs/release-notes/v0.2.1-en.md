# Renewlet v0.2.1

Renewlet v0.2.1 adds read-only access for automation and Telegram. You can create Public API tokens in Settings for external tools, and install Telegram Bot menu commands in a target chat to check status, the next renewal, and upcoming reminders without opening the web UI. Reminder emails also use a clearer summary and grouped layout, so larger reminder lists are easier to scan.

---

## New Features

- Public API: create read-only API tokens in Settings for Telegram Bot, CLI tools, Shortcuts, or automation platforms to query subscriptions, subscription details, status summaries, and upcoming due items.
- Telegram Bot commands: install commands into a target chat, then use the Telegram menu to check subscription status, the next renewal, today, this week, this month, and subscription summaries.
- Telegram message style: choose plain text or HTML rich text. Plain text has the broadest client compatibility; HTML rich text makes command results easier to read.

---

## Improvements

- Reminder emails now use a summary and grouped layout, making renewals, expiries, trial endings, and expired items easier to scan.
- Long reminder emails use a more compact presentation so large reminder lists are easier to open and read.

---

## Bug Fixes

- Fixed reminder emails that could repeat body content or make empty and test reminders unclear.
- Fixed unstable links between some form fields and their error messages in subscription, login, setup, reset password, and admin user forms.
