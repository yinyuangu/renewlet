# Renewlet v0.2.9

Renewlet v0.2.9 makes main-page navigation more stable across Docker and Cloudflare deployments, fixing occasional blank or flashing screens during route changes.

[中文 ->](./v0.2.9-zh.md)

---

## Read Before Upgrading

Cloudflare one-click deploy users should still upgrade from the generated repository by running `Sync Renewlet Upstream`. Do not click the one-click deploy button again. If an older generated repository does not have this entry yet, add it once from the [Cloudflare deploy documentation](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.md#existing-one-click-deploy-users).

---

## Improvements

- Moving between Dashboard, Subscriptions, Calendar, Statistics, and Settings is smoother, especially on slower networks.
- Statistics keeps the current view visible while exchange rates refresh, instead of sending the whole page back to a loading state.

## Fixes

- AI recognition drafts no longer treat renewal-date autofill as confirmation for other fields that still need review.
