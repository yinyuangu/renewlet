# Renewlet v0.2.1

Renewlet v0.2.1 增加了面向自动化和 Telegram 的只读查询能力。你可以在设置页创建 Public API token，让外部工具读取订阅状态；也可以把 Telegram Bot 菜单安装到指定聊天里，直接查看状态、下一条续费和近期提醒。提醒邮件也换成更清楚的摘要和分组布局，订阅较多时更容易扫读。

---

## 升级前请先读

Cloudflare 一键部署用户仍然从生成仓库运行 `Sync Renewlet Upstream` 升级，不要重新点一键部署按钮。旧生成仓库如果还没有这个入口，按 [Cloudflare 部署文档](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.zh-CN.md#旧一键部署用户) 一次性补上。

---

## 新功能

- Public API：可在设置页创建只读 API token，用于 Telegram Bot、CLI、Shortcuts 或自动化平台查询订阅列表、详情、状态摘要和未来到期项。
- Telegram Bot 查询命令：安装到指定聊天后，可以从 Telegram 菜单查询订阅状态、下一条续费、今天、本周、本月和订阅摘要。
- Telegram 消息样式：可选择纯文本或 HTML 富文本；纯文本兼容性最好，HTML 富文本让查询结果更清楚。

---

## 优化改进

- 提醒邮件改成摘要和分组布局，续费、到期、试用结束和已过期提醒更容易扫读。
- 长提醒邮件会使用更紧凑的呈现，避免大量项目让邮件难以打开或阅读。

---

## Bug 修复

- 修复提醒邮件正文重复、空提醒或测试提醒不够清楚的问题。
- 修复订阅、登录、初始化、重置密码和管理员用户表单里部分错误提示与字段关联不稳定的问题。
