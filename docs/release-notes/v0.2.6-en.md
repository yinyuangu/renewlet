# Renewlet v0.2.6

Renewlet v0.2.6 focuses on clearer statistics and a tighter mobile experience. Cost trends now show which subscriptions make up each month, and subscription, filter, and Settings screens take less space on phones.

[中文 ->](./v0.2.6-zh.md)

---

## Read Before Upgrading

Cloudflare one-click deploy users should still upgrade from the generated repository by running `Sync Renewlet Upstream`. Do not click the one-click deploy button again. If an older generated repository does not have this entry yet, add it once from the [Cloudflare deploy documentation](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.md#existing-one-click-deploy-users).

---

## New Features

- The Statistics trend chart now includes month details, showing which subscriptions make up upcoming charges and amortized monthly cost. Detail rows can open the subscription details dialog directly.

---

## Improvements

- Mobile dashboard stat cards are more compact, so key numbers no longer take up as much of the screen.
- Subscription lists, filters, public status pages, and Settings feel tighter and easier to scan on phones.
- Subscription card badges, dates, and payment metadata wrap more naturally, making start and renewal dates easier to view together.
- The MFA method picker is easier to tap and read on small screens.
