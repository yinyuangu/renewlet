# Renewlet v0.2.7

Renewlet v0.2.7 让订阅清单更好筛。现在可以按扣费周期、支付方式、币种、续费日期、置顶状态、公开页可见性和提醒状态组合查找，不用靠搜索框一条条翻。

[English ->](./v0.2.7-en.md)

---

## 升级前请先读

Cloudflare 一键部署用户仍然从生成仓库运行 `Sync Renewlet Upstream` 升级，不要重新点一键部署按钮。旧生成仓库如果还没有这个入口，按 [Cloudflare 部署文档](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.zh-CN.md#旧一键部署用户) 一次性补上。

---

## 新功能

- 订阅清单新增“更多筛选”，可组合扣费周期、支付方式、币种、续费日期范围、置顶、公开页可见性、提醒状态和重复提醒。
- 支付方式和币种筛选支持搜索、多选和摘要预览，选项多时不用反复滚动查找。

---

## 改进

- 已启用的筛选会集中显示，可单独移除，也可以一键清空。
- Docker 和 Cloudflare 部署下，筛选后的总数、翻页和结果保持一致，长列表查询更可靠。
