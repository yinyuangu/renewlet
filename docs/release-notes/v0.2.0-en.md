# Renewlet v0.2.0

Renewlet v0.2.0 brings language selection and AI recognition output under one user-facing language rule. First-time sessions choose the interface language from the browser or request context, saved language preferences follow the account, and AI recognition only localizes newly generated notes, categories, tags, and payment methods while preserving user input and existing configuration text. Settings and statistics also fix a few interface details that could interrupt everyday use.

[中文 ->](./v0.2.0-zh.md)

---

## Improvements

- First-time sessions choose the interface language from the browser or request context, with English as the fallback when nothing matches.
- Language switches in Settings are previews until you save; saved preferences are then stored on the account and follow other devices.
- AI recognition now generates new notes, categories, tags, and payment methods in the current language while preserving user input and existing configuration text.
- The header, settings sections, login, setup, and reset password pages are less likely to overflow or get covered on narrow screens.
- The personal cost basis toggle now sits beside the statistics overview title, and the loading skeleton uses the same placement.

---

## Bug Fixes

- Fixed AI recognition output that could follow example language instead of the current interface language.
- Fixed desktop Settings anchor jumps that could place sections too close to the sticky header.
- Fixed the statistics personal cost toggle shifting position after loading.
