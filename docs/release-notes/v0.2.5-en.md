# Renewlet v0.2.5

Renewlet v0.2.5 makes it faster to add similar subscriptions and cleans up the subscription card action menu in both English and Chinese.

[中文 ->](./v0.2.5-zh.md)

---

## Read Before Upgrading

Cloudflare one-click deploy users should still upgrade from the generated repository by running `Sync Renewlet Upstream`. Do not click the one-click deploy button again. If an older generated repository does not have this entry yet, add it once from the [Cloudflare deploy documentation](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.md#existing-one-click-deploy-users).

---

## New Features

- Subscription cards now include a Copy action for creating a new subscription from an existing one. Before saving, you can still adjust the name, price, dates, logo, tags, notes, cost sharing, pinning, and public visibility.

---

## Improvements

- The subscription card action menu now sizes to its content. Long English actions stay on one line, while shorter Chinese menus stay compact.
